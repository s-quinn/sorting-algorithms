# sorting
implementations of different sorting algorithms

## merge sort
* stable sort
* time complexity
  * averge case : O(nlogn)
  * worst case: O(nlogn) 
* space complexity
  * n
  
## quick sort
* in place
* time complexity
  * averge case : O(nlogn)
  * worst case: O(n<sup>2</sup>) 
* space complexity
  * logn
